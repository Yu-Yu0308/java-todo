# 📝 Java Todo アプリケーション

**⚠️ 学習目的で作成されたソフトウェアです**

シンプルで使いやすいTodo管理アプリケーションです。コマンドライン上で動作し、CSVファイルを使用してデータを永続化します。

**重要**: このソフトウェアはJavaの学習・リハビリ目的で作成されたため、テストやリファクタリングが行き届いていない場合があります。本番環境での使用は推奨されません。

## ✨ 機能

- **Todo追加**: 新しいタスクを追加し、完了状態を設定
- **Todo削除**: 不要なタスクを削除し、番号を自動振り直し
- **Todo編集**: 既存のタスクの内容と状態を変更
- **Todo表示**: すべてのタスクを一覧表示
- **データ永続化**: CSVファイルを使用してデータを保存
- **キャンセル機能**: 各操作をキャンセル可能

## 🚀 技術仕様

- **言語**: Java
- **データ形式**: CSV
- **エンコーディング**: UTF-8
- **動作環境**: Windows (バッチファイル対応)

### 前提条件
- Java 8以上がインストールされていること
- Windows環境（バッチファイルを使用する場合）
- **重要**: バックアップを取ってから使用することを強く推奨します

### インストール手順

1. リポジトリをクローン
```bash
git clone https://github.com/your-username/java-todo.git
cd java-todo
```

2. コンパイル
```bash
javac -encoding UTF-8 *.java
```

3. 実行
```bash
# Windowsの場合
run.bat

# または直接実行
java -Dfile.encoding=UTF-8 Todo
```

## 📖 使用方法

### 基本操作

1. **アプリケーション起動**
   ```
   予定一覧
   [未]   1：勉強
   [未]   2：研究
   [未]   3：アルバイト
   [完]   4：プログラミング
   
   アクションを1~4で選択してください。
   1: Add  2: Delete   3: Edit   4: Finish
   ```

2. **Todo追加**
   - `1`を選択
   - Todo名を入力
   - 状態を選択（0: 未完了, 1: 完了）

3. **Todo削除**
   - `2`を選択
   - 削除したいTodoの番号を入力

4. **Todo編集**
   - `3`を選択
   - 編集したいTodoの番号を入力
   - 新しい内容と状態を入力

5. **終了**
   - `4`を選択してプログラム終了

### キャンセル機能
各操作中に`0`を入力することで操作をキャンセルできます。

## 💡 設計の特徴

### オブジェクト指向設計
- **Todo.java**: メインアプリケーション、ファイルI/O、表示機能
- **Operate.java**: Todo操作（CRUD）のロジック
